# next_retail
Collecting and Analyzing data from Next Retail Website

# Project Brief:
* Stock Levels
    * Total SKU
    * Total SKU OOS
        * SKUs >= 1 OOS = OOS
    * % SKU OOS
* Sales
    * Total SKU on sale
    * % of SKU on sale
    * Average discount %
        * Histogram, bin ranges
        
## Definitions
* SKU - Stock-Keeping Unit.  Unique code for each individual item
* OOS - Out of Stock
